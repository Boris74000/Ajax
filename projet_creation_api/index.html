Bieronomy

Bonjour humble visiteur, et bienvenue sur Bieronomy !
Ce site vous propose de recenser les bières incontournable

Utilisation de l'API

Cette API vous permettra d'obtenir sous la forme d'un JSON un ensemble d'informations sur les bières.
Nom, Pays, Degrés, Amertume, Couleur, Prix.
L'url à utiliser est de la forme suivante "...?param1=value1¶m2=value"

Trier

Vous pouvez trier les bières grâce à trois paramètres :

  • Couleur
  • Pays
  • Degrés

Créer

Vous pouvez créer des nouvelles bières grâce à cette URL : "...?param1=value1¶m2=value"
Exemple :
nom=Duvel&pays=Belgique°res=9&amertume= nom :Duvel, Pays: Belgique, Degrés: 9, Amertume: 36, Couleur: Blonde, Prix: 2

Modifier

Vous pouvez modifier des bières existantes grâce à cette URL : "...?param1=value1¶m2=value"

Supprimer

Vous pouvez supprimer des bières grâce à cette URL : "...?param1=value1¶m2=value"

Afficher

Vous pouvez afficher des bières grâce à cette URL : "...?param1=value1¶m2=value"
