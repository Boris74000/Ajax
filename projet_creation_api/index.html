Bieronomy

Bonjour humble visiteur, et bienvenue sur Bieronomy !
Ce site vous propose de recenser les bières incontournables

Utilisation de l'API

Cette API vous permettra d'obtenir sous la forme d'un JSON un ensemble d'informations sur les bières.
Nom, Pays, Degrés, Amertume, Couleur, Prix.
L'url à utiliser est de la forme suivante "...?param1=value1&param2=value"

Trier

Vous pouvez trier les bières grâce à trois paramètres :

  • Couleur
  • Pays
  • Degrés

Créer

Vous pouvez créer des nouvelles bières grâce à cette URL : "...newbeer.php?param1=value1&param2=value2&param3=value3&param4=value4&param5=value5&param6=value6"
Exemple :
"...newbeer.php?nom=Duvel&pays=Belgique&degres=9&amertume=36&couleur=blonde&prix=2"

Modifier

Vous pouvez modifier une bière existante grâce à cette URL : "...update.php?param1=value1&param2=value2&param3=value3&param4=value4&param5=value5&param6=value6"
Exemple :
"...update.php?nom=Duvel&pays=Belgique&degres=9&amertume=36&couleur=blonde&prix=2"

Supprimer

Vous pouvez supprimer des bières grâce à cette URL : "...deletebiere.php?param1=value1"
Exemple :
"...deletebiere.php?id=5"

Afficher

Vous pouvez afficher la liste des bières avec leurs ID grâce à cette URL : "...bieres.php"

Selectionner

Vous pouvez selectionner une bière grâce à son ID et afficher son détail grâce à cette URL :"...une biere.php?param1=value1"
Exemple:
"...une biere.php?id=2"
